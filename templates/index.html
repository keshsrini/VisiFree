How to Use This Platform

  1. Visit the Settings page to customize your experience. Adjust font size, voice speed, color scheme, and audio cues.
  2. Head to the Design page to start creating your website.
  3. Use component buttons to add elements to your canvas, with voice guidance and audio cues.
  4. Navigate between components using keyboard shortcuts:
    • Press Enter to edit a component
    • Use Delete to remove a component
    • Reorder components with arrow keys
  5. Save your work with the Save Design button and retrieve it later using Load Design.

Every action on this platform is designed with accessibility in mind. If you need assistance, use your screen reader's shortcut keys to navigate and explore the page contents.
